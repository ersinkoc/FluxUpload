Simple, Powerful API

Get started with just a few lines of code. FluxUpload integrates seamlessly with Express, Fastify, Koa, or plain Node.js HTTP servers.

  • Plug & play with any Node.js framework
  • Built-in validation, hashing, and compression
  • Local storage and S3 out of the box
  • Full TypeScript definitions included
server.js 
const FluxUpload = require('fluxupload');
const { LocalStorage, QuotaLimiter,
        MagicByteDetector, StreamHasher } = require('fluxupload');

const uploader = new FluxUpload({
  validators: [
    new QuotaLimiter({ maxFileSize: 10 * 1024 * 1024 }),
    new MagicByteDetector({ allowed: ['image/*'] })
  ],
  transformers: [
    new StreamHasher({ algorithm: 'sha256' })
  ],
  storage: new LocalStorage({
    destination: './uploads',
    naming: 'uuid'
  })
});

// Express.js
app.post('/upload', async (req, res) => {
  const result = await uploader.handle(req);
  res.json(result);
});

Everything You Need

A complete file upload solution with security, performance, and observability built-in.

Zero Dependencies

Uses only native Node.js modules. No external packages means fewer vulnerabilities and smaller bundle size.

Stream-First Design

O(1) memory usage. Upload a 10GB file using only ~10MB RAM. Never buffers entire files in memory.

Security-First

Magic byte verification, DDOS protection, atomic writes, rate limiting, and CSRF protection built-in.

Plugin Architecture

Validators, Transformers, and Storage drivers. Extend functionality without modifying core code.

Multi-Storage Support

Local filesystem, AWS S3, MinIO, DigitalOcean Spaces. Upload to multiple destinations simultaneously.

Full Observability

Structured logging, Prometheus metrics, progress tracking, and health checks. Production-ready monitoring.

Built-in Plugins

Everything you need out of the box. No additional packages required.

Validators

  • QuotaLimiter Size limits
  • MagicByteDetector File type verification
  • ImageDimensionProbe Dimension checks
  • RateLimiter Request throttling
  • CsrfProtection CSRF token validation

Transformers

  • StreamHasher SHA256/MD5 checksums
  • StreamCompressor Gzip/Brotli compression

Storage Drivers

  • LocalStorage Filesystem storage
  • S3Storage AWS S3 & compatible

Works with MinIO, DigitalOcean Spaces, Backblaze B2, and more.
